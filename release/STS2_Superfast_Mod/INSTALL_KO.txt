STS2 Superfast Mod 설치 안내

1. Slay the Spire 2 설치 폴더에 `mods` 폴더를 만드세요.

   D:\Program Files (x86)\Steam\steamapps\common\Slay the Spire 2\mods

2. 아래 4개 파일을 `mods` 폴더에 넣으세요.
   - sts2-speed-skeleton.pck
   - sts2-speed-skeleton.dll
   - Sts2Speed.Core.dll
   - Sts2Speed.config.json

3. `Sts2Speed.config.json`을 메모장으로 열고 속도를 조절하세요.

기본 추천값:
{
  "enabled": true,
  "baseSpeed": 3.0,
  "spineSpeed": 1.0,
  "queueSpeed": 1.0,
  "effectSpeed": 1.0,
  "combatUiSpeed": 1.0,
  "combatVfxSpeed": 1.0,
  "combatOnly": true
}

핵심 설명:
- baseSpeed: 전체 기본 배속
- spineSpeed: Spine 애니메이션 계열 계수
- queueSpeed: 행동 사이 wait / 처리 템포 계수
- effectSpeed: timer 기반 effect delay 계수
- combatUiSpeed: 조준 / intent / 카운터 같은 전투 UI 계수
- combatVfxSpeed: 데미지 숫자 / trail 같은 전투 VFX 계수
- combatOnly: 전투 중에만 적용할지 여부

중요:
- JSON 숫자는 전부 "클수록 빠름" 입니다.
- 실제 적용 배속은 `baseSpeed x 각 항목 Speed` 입니다.
- 초보자는 `baseSpeed`만 바꾸고 나머지는 `1.0`으로 두는 것을 추천합니다.
- 기본 추천 시작값은 `baseSpeed = 3.0` 입니다.
- 가장 자연스러운 시작점도 `baseSpeed`만 조절하고 나머지는 `1.0`으로 두는 쪽입니다.

4. 게임을 실행하세요.

설정 변경 방법은 두 가지입니다.
- `mods\Sts2Speed.config.json` 파일을 직접 수정
- 게임 안 `설정 -> 모드 -> STS2 Speed Skeleton` 화면에서 직접 조절

처음에는 `baseSpeed`만 바꾸고 나머지는 그대로 두는 편이 가장 무난합니다.

추가 튜닝 설명:
- 더 자세한 설명은 저장소의 `docs\TUNING_KO.md` 문서를 참고하세요.

주의:
- 모드를 처음 켜면 진행 데이터가 비어 보일 수 있습니다.
- 이 경우 `AppData\Roaming\SlayTheSpire2\steam\<계정>\profile1` 내용을 `modded\profile1`로 복사하면 됩니다.
- 게임 실행 중에는 `mods` 폴더 파일을 교체하지 않는 편이 안전합니다.
