STS2 Superfast Mod 설치 안내

1. Slay the Spire 2 설치 폴더에 mods 폴더를 만드세요.
   예:
   D:\Program Files (x86)\Steam\steamapps\common\Slay the Spire 2\mods

2. 아래 4개 파일을 mods 폴더에 넣으세요.
   - sts2-speed-skeleton.pck
   - sts2-speed-skeleton.dll
   - Sts2Speed.Core.dll
   - Sts2Speed.config.json

3. Sts2Speed.config.json을 메모장으로 열고 속도를 조절하세요.

기본 예시:
{
  "enabled": true,
  "baseSpeed": 3.0,
  "spineSpeed": 1.0,
  "queueSpeed": 1.0,
  "effectSpeed": 1.0,
  "combatUiSpeed": 1.0,
  "combatVfxSpeed": 1.0,
  "combatOnly": true
}

설명:
- baseSpeed: 전체 기본 배속
- spineSpeed, queueSpeed, effectSpeed, combatUiSpeed, combatVfxSpeed: 항목별 계수
- 실제 속도 = baseSpeed x 각 Speed 값

중요:
- 이 파일의 숫자는 전부 클수록 빨라집니다.
- queueSpeed, effectSpeed도 내부적으로는 duration을 줄이는 방향으로 처리되어서 사용자는 숫자를 올리면 더 빨라진다고 이해하면 됩니다.
- 초보자는 baseSpeed만 바꾸고 나머지 Speed 값은 1.0 그대로 두면 됩니다.

예:
- 3.0 = 현재 기본 추천값
- 2.0 = 비교적 가벼운 배속
- 1.0 = 바닐라와 비슷함

4. 게임을 실행하세요.

주의:
- 모드를 처음 켜면 진행 데이터가 비어 보일 수 있습니다.
- 이 경우 AppData\Roaming\SlayTheSpire2\steam\<계정>\profile1 내용을 modded\profile1로 복사하면 됩니다.
- 게임 실행 중에는 mods 폴더 파일을 교체하지 않는 편이 안전합니다.
